Create a New Set

As an IFPA member, you may create new "sets" within the video or photo galleries on footbag.org. Simply read the instructions next to each field in the form below, and then fill out the form.

Note: you'll be able to add photos or videos to the new set once it's created. Just follow the instructions as you go.

Set Keyword: Choose a unique and relevant keyword which will be the primary name for this set. Note: keywords must only be one word and must contain no spaces, and no special characters. This is the short name of your set which will go in web-based addresses; so like an e-mail address, it can't have spaces or special characters in it! Try to think of something really original; like an e-mail address on hotmail. :-)
Contents: Sets can contain either other sets, or individual items (photos or videos). If this set has media in it (photos or videos), then mark is as containing "individual items". Else, you can create hierarchies by defining sets that contain other sets.
Container: You can create your own hierarchies by putting sets within other sets (called "containers"). If you don't have any other containers in the pull-down menu, you may want to make this set a container (by setting the "Contents" above to "other sets") and then add a new set to it (on the next screen).
Enabled: You can hide this set so that it won't be found by other users, if you're not ready to have it appear in a list of available sets. But we recommend leaving it "visible" so you don't forget. (You'll always be able to see this, and setting it to "hidden" doesn't hide it completely -- if someone knows the keyword above, they can see the set. But making it "hidden" keeps it from appearing in the index of available galleries.)
Type: Gallery sets can be either collections of photos or collections of videos (but not both at the same time). So, choose which type this set is, and stick to it.
Title: Title to display for this set of images or videos. Try to be as descriptive as possible, e.g., "Photos from the 2005 World Footbag Championships".
Author: Credits for this set of images or videos. Note that you can override these credits for any individual items in the set, by editing those items directly.
Copyright: Year and owner if applicable (e.g., '1997, Oriam Sports'). Note that you can override this copyright information for any individual item in the set by editing the item directly.
Info: Text describing this set, to appear wherever the complete set is displayed.
Related page: Reference to related page (URL); e.g., if this is a set of pictures from a tournament, put the URL of the tournament page here
