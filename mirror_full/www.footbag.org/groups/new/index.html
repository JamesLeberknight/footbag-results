Add IFPA Group/Committee

Fill out this form to describe the new group or committee:

Name:
This is the full name of your group or committee. Please make it as descriptive as possible, i.e., "IFPA Membership Committee", "IFPA Working Group for Freestyle Videos", etc.
Policy: By setting this group to "public", users will be able to find this group by going to the IFPA group list. If you leave it "private", it will not be automatically listed.
Type: Is this an official IFPA Committee, an ad-hoc "group"? (There's no difference in terms of how the system works; only IFPA members can use this feature. But it helps to describe the group correctly.) Try to pick the most accurate description -- use "committee" or "group" if unsure.
Restrict
Membership:
Important: you can create a group that is open to all members, or just to members of a specific existing group or committee (i.e., a subcommittee). If you want to constrain your membership to only a particular existing group (of which you must be a member), please pick that group from the pull-down, which lists your current group/committee memberships.
Description:
This will be displayed on your group's automatic "home page". Please be sure to verify your spelling, punctuation, and to make this as formal as possible. As with any content on footbag.org, we reserve the right to remove this group if it doesn't conform to IFPA website policies. Keep it as professional and serious as possible. Thanks.
Keyword: Unique, one-word name of the group, used for auto-email (if enabled) and also for simpler web page address for auto-generated group home page. Please make this one short word, no spaces or punctuation. Normally this is just the letters that make up the abbreviation of the group title. Examples: woc, ifc, fvids, etc.
Email: Do you want an automatic e-mail list to be created for this group? The list address will be whatever you typed in the keyword field above @ifpa.footbag.org. In other words, if you typed "fvids" in the Keyword field, then this list would be "fvids@ifpa.footbag.org". Note: this is not a moderated discussion list. Those are different. This is fully automated and is tied directly to this group's membership. There are no archives (yet) or other services (such as moderation). What you see is what you get.
Email Options:  Add Subject?
Do you want footbag.org to automatically insert a special keyword or phrase (such as [groupname]) on the front of each subject line whenever anyone posts a message to this group's e-mail list? (Note: the e-mail option has to be turned on above for this to work.) If so, click the check box and either use the default value provided, or modify the text to have it say what you want. Note that you should try to keep this very short (like the group keyword), otherwise message subject lines could be very long. This feature makes it easy for people without filters to keep track of mailing list discussions. However, it is not fool-proof and should be used with caution. A better solution is simply to filter your incoming e-mail based on the To: or Cc: line of mail you receive; if it's sent To: groupkeyword@ifpa.footbag.org or Cc: groupkeyword@ifpa.footbag.org, you can filter it into a special folder on your hard drive for easy perusal.
 Restricted?
 Do you want to make this e-mail list "closed" so that only members of this group can send e-mail to this list? If you check the check-box, only people who send e-mail from their registered footbag.org e-mail address will be able to send messages to this group. This may be a problem if you have members with multiple e-mail addresses. By checking this option, anyone not considered to be in the group will get an error message if they try to e-mail this group.
